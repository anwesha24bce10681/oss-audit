## Project Description
This project is an audit of the open-source software Git. It covers its origin, license, ethical aspects, Linux usage, and comparison with proprietary tools. The project also includes five shell scripts demonstrating basic Linux concepts.

## Scripts Included

### 1. System Identity Report
Displays system information such as kernel version, user, uptime, and date.

### 2. FOSS Package Inspector
Checks whether Git is installed and shows its version and details.

### 3. Disk and Permission Auditor
Analyzes important directories and displays their size, permissions, and ownership.

### 4. Log File Analyzer
Reads a log file and counts the number of error entries.

### 5. Open Source Manifesto Generator
Takes user input and generates a personalized open-source philosophy statement.
